## Как устроен пакет

У пакета два уровня API:

- high-level API  
  Это основной и самый удобный вариант. Ты работаешь через `Finam::connect($token)` и вызываешь понятные методы вроде `getLatestQuotes()` или `getOrders()`.
- low-level API  
  Это прямой доступ к REST-клиенту. Он нужен, если ты хочешь сам управлять resource wrappers или работать ближе к HTTP.

Если не уверен, с чего начать, используй high-level API.
